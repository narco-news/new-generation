# narco.news

The source code for the [narco.news](https://narco.news) website. It's built with Vue 3
and deployed to Vercel as a serverless app. It's a mixture of this [example](https://github.com/kadiryazici/vite-ssr-vue3-example)
and the [Vitesse SSR](https://github.com/frandiox/vitesse-ssr-template) template.
